Curso HTML y CSS

En este curso aprenderás los fundamentos del desarrollo web, incluyendo HTML y CSS. Aprenderás a crear páginas web estáticas y a darles estilo con CSS, para que puedas diseñar sitios web atractivos y funcionales.

Descripción del Curso

Este curso está diseñado para principiantes que desean aprender a crear páginas web desde cero. Aprenderás a usar HTML para estructurar el contenido y CSS para darle estilo y diseño. Al finalizar, serás capaz de crear páginas web responsivas y atractivas.

Temas Principales

  • Introducción a HTML: etiquetas, atributos y estructura básica.
  • Uso de CSS: selectores, propiedades y estilos.
  • Diseño responsivo con Flexbox y Grid.
  • Buenas prácticas de maquetación y accesibilidad.
  • Proyecto final: Creación de un portafolio personal.

Duración

4 semanas (20 horas estimadas).
