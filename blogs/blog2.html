¿Qué es CSS?

CSS (siglas en inglés de Cascading Style Sheets; en español: «hojas de estilo en cascada») es un lenguaje informático especializado en definir y cohesionar la presentación de un documento escrito en un lenguaje de marcado como HTML o XML; por ello, puede ser aplicado a cualquier documento etiquetado en XHTML, SVG, XUL, RSS, etcétera.2​ Junto con el propio HTML y JavaScript, CSS está en la base de la tecnología usada por muchos sitios web para crear páginas visualmente atractivas, interfaces de usuario para aplicaciones web y GUIs para aplicaciones móviles.3​

Historia de CSS

CSS fue propuesto por primera vez por Håkon Wium Lie el 10 de octubre de 1994.9​ Al mismo tiempo, Lie trabajaba con Tim Berners-Lee en el CERN.10​ Muchos otros lenguajes de hojas de estilos fueron propuestos al mismo tiempo, y las discusiones en las listas de correo públicas dentro del W3C dieron lugar a la primera Recomendación CSS por el W3C (CSS1)11​ en 1996. En particular, la propuesta de Bert Bos fue influyente; él fue el coautor de CSS1 y es reconocido como el cocreador de CSS.

Elementos de CSS

CSS se compone de una serie de reglas que se aplican a los elementos de un documento HTML. Cada regla se compone de un selector y un bloque de declaración. El selector indica a qué elementos se aplicará la regla, y el bloque de declaración contiene las propiedades y valores que se aplicarán a los elementos seleccionados. Algunos de los elementos más comunes de CSS son:

  • Color: Define el color del texto.
  • Font-size: Define el tamaño de la fuente.
  • Font-family: Define la fuente utilizada para el texto.
  • Background-color: Define el color de fondo de un elemento.
  • Border: Define el borde de un elemento.
  • Margin: Define el margen exterior de un elemento.
  • Padding: Define el relleno interior de un elemento.
